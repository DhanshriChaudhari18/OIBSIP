# 📊 Sales Prediction

## 📌 Description
This project predicts product sales based on advertising expenditure on TV, Radio, and Newspaper.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Algorithm
- Linear Regression

## 📊 Result
- Mean Absolute Error: ~1.5

## 📷 Output
![output.png](output.png)

## 📊 Insights
- TV advertising has the highest impact on sales.
- Radio also contributes significantly.
- Newspaper has less impact.

## 🚀 Conclusion
The model helps in predicting future sales and can assist businesses in optimizing advertising budgets.
